# Bliss Marketplace — Homepage Prototype

Interactive prototype for the new blisswellnesscare.com consumer homepage, restructured around the Fresha model: search-first marketplace for wellness and sports services in Hong Kong.

**Live preview:** https://whhoaf.github.io/bliss-homepage-prototype/

## What's in the prototype

- Search-first hero: service/name + district + time, working text filter
- Category chips and three vertical tiles (Wellness / Sports / Hybrid)
- 9 sample vendor cards with ratings, districts, from-prices, promo tags
- "Pick your coach" tag on sports listings (book-by-coach concept)
- Location-based discovery: illustrative HK map with clickable district pins that filter listings
- How-it-works, community reviews, stats line
- Bliss for Business strip (self-serve listing pitch)
- Full EN / 繁 language toggle (top nav)
- Mobile-first responsive layout

## Notes for reviewers

- All listings, reviews and stats are **sample data**.
- Map is illustrative; production build gets a live map with real-time availability.
- Design follows the Bliss design system (periwinkle + ink + indigo, serif display, pill shapes).
- Single self-contained `index.html`, no build step. Open locally or via the live preview link.
